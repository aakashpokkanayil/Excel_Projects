# Excel Project: Personal Income, Expense Tracker

## Overview
This is an Pratice Excel Project which contains sample monthly budget figures and its annual calculations

## Features
- **Tables:** Implemented tables for efficient data organization and management and automatic total yearly monthly calculations.
- **Auto Fill Options:** Utilized auto fill options for quick and accurate data entry.
- **Alignment & Formatting:** Fine-tuned alignment and formatting for improved readability and presentation.
- **Conditional Formatting:** Highlighted important insights using conditional formatting to identify trends and outliers. Here yearly saving which less then 40k is filled as red.
- **Calculation Functions:** Streamlined calculations with Auto SUM for accurate analysis.
- **Borders & Formatting:** Added professional touch with borders and formatting to enhance the visual appeal.
- **Layout Optimization:** Optimized page layout for both digital and print viewing for easy accessibility.

## Usage
1. Download the Excel file from the repository.
2. Open the Excel file in Microsoft Excel or any compatible spreadsheet software.
3. Explore the different sheets for Personal Income, Expense Tracker.
